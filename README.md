# Открытие и чтение файла, запись в файл
## Задача №1
Создание словаря с названием блюд и их ингредиентами.
## Задача №2
Реализована функция, которая показывает какие ингредиенты их количество, нужны для создания определеннёго блюда из списка. Так же программа расчитывает это с учётом количества персон. При запуске данного кода, будет создан файл под названием "dishes.txt", в который запишется информация о блюдах из кулинарной книги и будет возвращён результат функции в консоль.
